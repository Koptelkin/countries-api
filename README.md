SPA-приложение. 
Позволяет найти и отобразить в списке стран интересующую с помощью инпута и селекта. 
Использует api: 'https://restcountries.com/v2/all?fields=name,capital,flags,population,region'. 

stack:
1) create-react-app
2) react + typescript
3) redux
4) react-router
5) css: styled-components
